#  SoccerStat

### SoccerStat представляет собой приложение для просмотра спортивной статистики.

## Основные функции
- Просмотр списка лиг 
- Просмотр списка команд
- Просмотр календаря лиги с фильтром по дате
- Просмотр календаря команды с фильтром по дате
- Поиск лиг и команд по названию

## Используемые технологии
- React 19
- TypeScript 5.9
- Vite 7
- Ant Design 6
- React Router 7
- Testing Library
- ESLint 9

## Инструкция по запуску приложения: 
1) Перейти в папку soccer-stat и выполнить команду для установки зависимостей:

```bash
    npm install
```
2) В папке soccer-stat для запуска приложения в dev-режиме выполнить команду: 
```bash
    npm run dev
```
3) В браузере открыть http://localhost:5173/ и начать пользоваться приложением.

## Запуск тестов
Для запуска тестов в папке soccer-stat нужно выполнить команду:
```bash
    npm run test
```
## Запуск линтера
Для запуска линтера в папке soccer-stat нужно выполнить команду:
```bash
    npm run lint
```
